# compara.ai

Comparador de ativos da B3 — Ações e FIIs com indicadores fundamentalistas, ranking automático e análise visual.

## Features
- 📈 Comparador de Ações (12 indicadores R2A)
- 🏢 Comparador de FIIs (11 indicadores)
- 🔒 Bloqueio setorial automático
- 🏆 Ranking por duelos com pódio e radar
- ⓘ Tooltips educativos em cada indicador
- 👤 Cadastro com validação (CPF, e-mail, celular)
- 📋 LGPD compliance
- 🔐 Painel admin com métricas de usuários
- 📱 PWA instalável no celular

## Tech Stack
- React 18 + Vite
- Recharts (gráficos)
- PWA (vite-plugin-pwa)
- LocalStorage (dados)

## Deploy
```bash
npm install
npm run build
```

Deploy automático via Vercel conectado ao GitHub.
